# 🎬 Script de démonstration vidéo — ImageCaption AI
# Durée cible : 45–60 secondes

---

## SCÈNE 1 — Introduction (0:00 – 0:08)
**À l'écran :** Page d'accueil de l'application ouverte dans le navigateur.

🎙️ **Dire :**
> "Voici ImageCaption AI, mon projet de fin de module IA.
> Cette application génère automatiquement une description textuelle
> de n'importe quelle image grâce au modèle BLIP de Salesforce."

---

## SCÈNE 2 — Upload d'une image (0:08 – 0:18)
**À l'écran :** Faire un drag-and-drop d'une photo (chien, paysage, repas).
Montrer la prévisualisation qui s'affiche.

🎙️ **Dire :**
> "Il suffit de glisser une image dans la zone d'upload...
> La prévisualisation s'affiche immédiatement."

---

## SCÈNE 3 — Génération (0:18 – 0:35)
**À l'écran :** Cliquer sur le bouton vert "Générer la légende".
Montrer l'animation de chargement, puis la légende qui apparaît.

🎙️ **Dire :**
> "On clique sur Générer... Le modèle analyse l'image côté serveur
> grâce à un encodeur Vision Transformer et un décodeur de texte.
> En quelques secondes, la légende est prête !"

---

## SCÈNE 4 — Résultat & actions (0:35 – 0:48)
**À l'écran :** Mettre en valeur la légende affichée.
Cliquer sur "Copier" puis montrer le bouton "Télécharger".
Cliquer sur "Nouvelle image" pour montrer le reset.

🎙️ **Dire :**
> "La description peut être copiée ou téléchargée en .txt.
> On peut recommencer avec n'importe quelle autre image."

---

## SCÈNE 5 — Conclusion (0:48 – 0:58)
**À l'écran :** Afficher le terminal avec le log de démarrage FastAPI.
Finir sur la page GitHub du projet ouverte.

🎙️ **Dire :**
> "Le backend tourne sur FastAPI, le modèle sur CPU ou GPU.
> Tout le code est disponible sur GitHub — lien dans la description."

---

## 💡 Conseils de tournage
- Résolution : 1920×1080 minimum
- Outil recommandé : OBS Studio (gratuit) ou Loom
- Choisissez des images visuellement parlantes (animaux, nourriture, sport)
- Montrez 2 ou 3 images différentes pour démontrer la généralisation
- Fond neutre ou IDE visible = aspect professionnel
